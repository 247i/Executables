  ЛИЦЕНЗИОННОЕ СОГЛАШЕНИЕ

  Настоящее лицензионное соглашение является юридическим соглашением 
  между конечным пользователем и автором данного программного 
  обеспечения (ПО) Дмитрием Сидоровым. Используя ПО, Вы принимаете 
  условия Соглашения.

  1. ПРЕДОСТАВЛЯЕМЫЕ ПРАВА

  1.0. РЕДАКЦИЯ, КОЛИЧЕСТВО КОПИЙ ЛИЦЕНЗИИ И ПРАВА

  Объем предоставленных прав зависит от РЕДАКЦИИ (типа лицензии), 
  количества поддерживаемых семейств ОС и количества КОПИЙ ЛИЦЕНЗИИ. 
  В окне "О программе" (меню ПО "Справка") можно увидеть РЕДАКЦИЮ, 
  количество КОПИЙ, срок лицензии. Если количество КОПИЙ не указано, 
  то предоставлена одна (1) лицензия. Если срок действия лицензии не 
  указан, то он не ограничен.

  1.1. РЕДАКЦИЯ "FREE EDITION"

  К редакции "FREE EDITION" применяются только пункты 1.1, 5 и 6.

  "FREE EDITION" можно использовать в некоммерческих целях, включая 
  образовательные цели учреждений и ознакомление. В редакции "FREE 
  EDITION" по одному запросу можно восстановить до 4000 файлов из 
  файловой панели (из выбранной директории без обработки вложенных 
  директорий).

  Копии "FREE EDITION" можно создавать и распространять при выполнении 
  всех следующих условий: ПО получено непосредственно от автора или с 
  веб-сайтов ПО; ключи и иные компоненты других РЕДАКЦИЙ не передаются; 
  состав и компоненты ПО не изменены (за исключением обратимого 
  перекодирования); происхождение, авторство, условия данного 
  Соглашения доступны конечному пользователю в явном виде. Копии FREE 
  EDITION могут быть включены в бесплатные и платные комплексы.

  1.2.1. РЕДАКЦИИ "EXPRESS" И "PERSONAL": ТОЛЬКО ЛИЧНОЕ ИСПОЛЬЗОВАНИЕ

  Редакции "EXPRESS" и "PERSONAL" дают право использовать ПО только в 
  личных некоммерческих целях. Запрещается любое использование ПО, 
  которое прямо или косвенно приводит к получению прибыли.

  1.2.2. РЕДАКЦИЯ "STANDARD": ИСПОЛЬЗОВАНИЕ ТОЛЬКО ДЛЯ СОБСТВЕННЫХ НУЖД

  Редакция "STANDARD" дает право использовать ПО физическим лицам и 
  организациям только для собственных нужд. Организации, сотрудники и 
  другие лица, предоставляющие услуги по восстановлению данных, 
  не могут использовать ПО.

  1.2.3 РЕДАКЦИЯ "PROFESSIONAL": КОММЕРЧЕСКОЕ ИСПОЛЬЗОВАНИЕ И ПЛАТНЫЕ 
  УСЛУГИ

  Редакция "PROFESSIONAL" дает право использовать функционал ПО как в 
  некоммерческих, так и в коммерческих целях, а также оказывать услуги 
  по восстановлению данных.

  1.3. ЗАПУСК И ХРАНЕНИЕ КОПИЙ ПО

  Вы можете ЗАПУСКАТЬ столько КОПИЙ ПО (иметь столько незавершенных 
  процессов), сколько КОПИЙ ЛИЦЕНЗИИ вам предоставлено.

  С учетом резервных копий, вы можете создавать и хранить на две (2) 
  копии ПО больше, чем количество КОПИЙ ЛИЦЕНЗИИ. В целях хранения одна 
  копия может включать несколько различных версий ПО в одном хранилище 
  данных.

  2. АКТИВАЦИЯ

  Активация разблокирует полную функциональность ПО. При активации на 
  сайты активации ПО отправляется неуникальный код, идентифицирующий 
  аппаратное или портативное устройство, для получения кода активации.

  С лицензией на одно семействo ОС ("1-OS") можно активировать только 
  версии ПО, предназначенные для того же семейства ОС, которое было 
  выбрано при первой активации. Лицензия "Multi-OS" дает право 
  активировать версии ПО для разных семейств ОС.

  По истечении срока лицензии вы можете продолжать ЗАПУСКАТЬ ПО в той 
  же среде, в которой оно было активировано, но ни поддержка, ни 
  повторная активация, ни гарантии функциональности не предоставляются.

  2.1. СТАЦИОНАРНАЯ АКТИВАЦИЯ привязывает ЗАПУСК КОПИИ ПО к конкретной 
  аппаратной конфигурации компьютера.

  Активация может быть отклонена, если вы меняете более трех (3) 
  связанных аппаратных конфигураций в год на одну КОПИЮ ЛИЦЕНЗИИ. Смена 
  процессора или материнской платы, использование дополнительной 
  виртуализации означают смену конфигурации. Не учитываются другие 
  изменения среды запуска, которые также могут потребовать повторной 
  активации ПО.

  2.2. ПОРТАТИВНАЯ АКТИВАЦИЯ (только в РЕДАКЦИИ "PROFESSIONAL") 
  привязывает ЗАПУСК КОПИИ ПО к подключенному портативному устройству.

  Предназначена для ЗАПУСКА КОПИИ ПО на разных компьютерах путем 
  подключения привязанного портативного устройства. Активация может 
  быть отклонена, если вы привязываете КОПИЮ ЛИЦЕНЗИИ более чем к 
  трем (3) устройствам в год.

  2.3. ОДНОРАЗОВАЯ АКТИВАЦИЯ (только в РЕДАКЦИИ "PROFESSIONAL") - 
  с использованием одноразовых паролей - позволяет дополнительно 
  ЗАПУСКАТЬ КОПИЮ ПО до пяти (5) раз в месяц на различных компьютерах.

  3. ОБНОВЛЕНИЯ ПО

  Если поддержка версии ПО прекращается, Вы можете использовать 
  следующую доступную версию ПО, при этом новое соглашение заменяет 
  предыдущее только в пунктах, уточняющих условия и явно расширяющих 
  Ваши права. В иных случаях новые версии могут использоваться только 
  полностью на условиях новых соглашений.

  Вы также можете использовать предыдущие доступные версии ПО на 
  условиях данного Соглашения. Однако идентификация аппаратной 
  конфигурации может отличаться и повлечь дополнительные ограничения на 
  количество изменений конфигураций в соответствии с пунктом 2.

  4. АПГРЕЙД ПО

  Апгрейд - изменение комплектации, числа копий лицензии или РЕДАКЦИИ. 
  Для апгрейда ПО необходимо иметь исходную лицензию, для которой 
  предоставляется апгрейд. При апгрейде новое лицензионное соглашение 
  заменяет собой предыдущее и Вы больше не можете использовать ключи 
  и иные компоненты, предоставленные по условиям предыдущей лицензии.
